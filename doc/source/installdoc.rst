#######################
Installing SeriesMarker
#######################

The easiest way to install SeriesMarker, is to use the binary distribution
that matches your operating system. If an :ref:`installer <binary_installer>`
is not available for your operating system, see
:ref:`Building from Source <building_from_source>` for guidelines,
how to build SeriesMarker for your system.
